Important Audio Preload Examples

Preload="auto" (Default)

Browser will load the entire audio file when the page loads:

Preload="metadata"

Only loads metadata (like duration) but not the audio file itself:

Preload="none"

Won't load anything until the user clicks play:

Notes about preload attribute:

  • auto - Browser should load the entire audio file
  • metadata - Browser should load only metadata
  • none - Browser should not preload anything
  • Note: Browsers may ignore preload settings based on user settings or network conditions
