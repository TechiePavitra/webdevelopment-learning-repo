Comprehensive Font Properties (live examples)

Each example shows the property name and a live preview. Read the meta text to see the property and typical usage.

font-family: 'Poppins', 'Segoe UI', Tahoma, sans-serif;
This text uses a custom font stack (font-family).
font-size: 24px;
This text is larger using font-size.
font-weight: 700; (bold)
This text is bold using font-weight.
font-style: italic;
This text is italic using font-style.
font-variant: small-caps;
Small Caps: This is an example of small-caps.
font-stretch: expanded;
This text uses font-stretch (wider glyphs when supported).
font-kerning: normal; (controls kerning pairs)
Kerning normal - AVWA practice to show pair spacing.
font-feature-settings: "liga" 1, "kern" 1;
Enables typographic features (liga, kern) when available.
font-optical-sizing: auto;
Optical sizing lets variable fonts adjust for size (if supported).
font-variation-settings: 'wght' 600;
Adjust variable font axes (weight) when using variable fonts.
font: italic small-caps 700 18px/1.4 'Poppins', Arial, sans-serif; (shorthand)
Shorthand combines style, variant, weight, size/line-height and family.

text-transform: uppercase;

Underline with color and style using text-decoration.
