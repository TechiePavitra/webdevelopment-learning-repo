Implementation of CSS in HTML

Inline CSS

This method is the least common and not advised for production use.

Example

                
                    <p style="background-color: aqua;">Hello World!</p>

Internal CSS

This method is common among non-professionals; you can use it for small CSS snippets, but it is not recommended for production-level websites.

Example

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Document</title>
                        <style>
                        h1 {
                            background-color: red;
                            color: white;
                        }
                        </style>
                    </head>

External CSS

This method is the most common and advised for production use; create a separate style.css file and link it as shown below.

Example

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Document</title>
                        <link rel="stylesheet" href="style.css">
                    </head>
