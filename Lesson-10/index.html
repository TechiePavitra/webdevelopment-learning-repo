What Are Semantic Elements (Tags) in HTML?

Semantic HTML is the use of HTML tags that clearly define the purpose of the content they enclose to make your website code more understandable for both developers and search engines (SEO).

These HTML tags are also called semantic elements. They provide a meaningful description of the content's role within a webpage.

For example, <header> is a semantic HTML tag. It encloses the header section (introductory content) at the top of the page.

Common HTML Semantic Tags (brief)

Semantic tags give meaning to page structure. Below are common semantic elements and their purpose:

<header>

Contains introductory content or navigation for a page or a section (for example: logo, site title, and heading).

<nav>

Holds a set of navigation links for the site or a section (main menu, table of contents).

<main>

Designates the primary content unique to the document. There should be only one <main> element per page.

<article>

Represents a self-contained composition that could be distributed independently (blog post, news article, forum post).

<section>

Groups related content together, typically with a heading — a thematic grouping within a page.

<aside>

Contains tangentially related content (sidebars, callouts, related links) that is separate from the main flow.

<figure> & <figcaption>

<figure> holds media (images, diagrams); <figcaption> provides a caption or legend for that media.

<footer>

Contains footer information for the page or section such as copyright, contact links, and legal notices.

Other useful semantic elements

  • <details> — a disclosure widget that the user can open/close.
  • <summary> — the summary/title shown for a <details> element.
  • <time> — represents a specific time or date.
  • <mark> — highlights text for relevance (search matches, important parts).
  • <address> — contact information for the author/owner of the page or section.

Using semantic tags improves accessibility (screen readers), SEO, and makes your markup easier to understand and maintain.
